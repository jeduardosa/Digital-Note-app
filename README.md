- **Linguagem**: Kotlin 100% interoperável
- **Arquitetura**: MVVM com Clean Architecture
- **Persistência**: Room Database
- **UI**: XML
- **Navegação**: Navigation Component
- **Padrões**: Coroutines, Flow, State Management
